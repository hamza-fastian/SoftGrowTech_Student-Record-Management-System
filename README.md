# SoftGrowTech_Student-Record-Management-System
## Student Record Management System

The Student Record Management System is a console-based application developed in C++ that helps manage student information efficiently. The project allows users to add, display, and update student records through a simple menu-driven interface. Each student record contains a student’s name, roll number, and marks. The program uses structures to organize data and dynamic memory allocation to store multiple student records during runtime.

This project is designed for beginners who want to learn the fundamentals of C++ programming. It demonstrates important programming concepts such as structures, functions, loops, conditional statements, pointers, arrays, and input validation. The application also handles invalid user input to improve reliability and user experience.

The add feature allows users to enter details for multiple students dynamically. The display feature prints all stored student records in a well-formatted layout. The update feature enables modification of existing records by searching through either the student’s name or roll number. Proper validation ensures that marks cannot be negative and invalid data types are rejected.

The system is lightweight, easy to understand, and suitable for educational purposes. It provides practical experience in building menu-driven console applications using standard C++ libraries. Although the current version does not support permanent data storage, file handling, or delete functionality, it serves as a strong foundation for more advanced student management systems in the future.
